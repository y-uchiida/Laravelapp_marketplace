# Laravelapp_shoppingCart
Laravel 8系でカート・決済機能付きECサイトを実装しました。  

# 開発環境
- Windows 11 Pro
- WSL2 Ubuntu 20
- Laravel Sail
- docker-compose version 1.29.2
